駭客手法 1:生活郵件偽裝 ( 釣魚郵件 Phishing)

駭客喜歡偽裝成「好康優惠」、「物流通知」或「罰單未繳」，利用您的好奇心或恐慌感。

駭客手法 2:偽裝寄件者 ( 變臉詐騙 BEC)

「顯示名稱」是可以被隨意修改的。駭客常將名稱改為長官或同事的姓名，但實際寄件地址卻是外部免費信箱。

公司正式網域結尾必為 @taisugar.com.tw。若看到 @gmail.com 或 @outlook.com 寄來的公務信，請直接刪除。

駭客手法 3:追蹤圖片( 網路信標 Web Beacons )

郵件中的「外部圖片」一旦載入，駭客就能取得您的 IP 位址與開信時間。

Q:事業部產品推廣信件也有圖片，為什麼不會被阻擋?
A:若圖片是「隨信內嵌」(概念上接近附件)，不需要連到外部網站讀取，就不會產生外部追蹤風險，因此不會被郵件軟體阻擋，可正常顯示。
